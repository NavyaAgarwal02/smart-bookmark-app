# Smart Bookmark App

A real-time bookmark manager built using Next.js and Supabase.

## Features
- Google OAuth login
- Private bookmarks
- Real-time sync
- Responsive UI
- Deployed on Vercel

## Tech Stack
- Next.js (App Router)
- Supabase
- Tailwind CSS

## Problems & Solutions

1. Google Auth Redirect
Solved using redirectTo option.

2. Real-time Sync
Used Supabase Realtime channels.

3. Private Data
Filtered bookmarks using user_id.
